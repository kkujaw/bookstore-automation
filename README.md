API Automation Testing Assessment: Online Bookstore - Java Implementation

# Bookstore API Automation

## Prerequisites

- Java 17+
- Maven 3.8+

## Setup

```bash
git clone <repo-url>
```

```bash
cd bookstore-api-automation
```

```bash
mvn clean install
```

## Running Tests

```bash
mvn test
```

## Test Reports

- Reports are generated in `target/surefire-reports/`
